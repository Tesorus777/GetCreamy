# Instructions on how to copy a file between Solution Projects
#### Open Markdown Preview with Shift + F7

## This is used for files such as DataAccessOptions.json

1. Right click the desired destination project
2. Select Add > Existing Item
3. Locate the desired file from the source project directory
    a. If necessary, select "All Files ("\*.\*") from dropdown
4. Click the dropdown arrow next to "Add" and select "Add as Link"
5. Locate the newly added file in the destination project
    a. Right click > Properties
    b. Select Copy to Output Director => Change to "Copy Always"
